WP Last Updated Date is a Plugin will show Date of your post's last updated of modified date.

== Description ==

Let's know your Blog post Reader that you've updated your post. WordPress Last Updated Date is necessary to keep Post/Page engagement with your audience. You have noticed that many blogs show the updated date instead of published date. This WordPress Plugin will allow you to do it automatically. No need to copy and paste any code into your Theme files. You can translate the "Last Update" word with your own language or can customize the text as you need.


It's been Super easy to use WP Last Updated Date. After activating WP Last Updated Date Plugin simply show Last Updated Date after Title and before Content.

You'll get an Option under Settings>WP Last Updated Date. You can Change "Last Updated" Text as your own Language.

Post Updated Date or Modified Date will be great feature for Google Search Engines. This Plugins will solve this issue very nicely. You don't have to worry about anything.

This Plugin comes for Posts Only, Pages, Products and Other Post Types functionality will come on updates.

== Installation ==

*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'post-modified-date.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation.
*  Activate the plugin.
*  With the activation a you'll see a Menu name "WP Last Updated Date" under Settings Menu.
